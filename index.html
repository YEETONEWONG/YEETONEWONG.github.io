# 白云山一日游 ## 早上九点前出发 建议8点半-9点出发，早上比较清爽； ### 交通 #### 五山-白云山 * 打车：从汕头校友楼到南门游客中心17-20元左右，等车时间4分钟，路程时间10-20分钟，人均5-7元左右； #### 国际校区-白云山 * 校巴+打车：乘坐校巴，八点五十到达五山校区，选择图书馆下车，直接在南门打车去往白云山； * 直接打车：在国际校区打车去白云山南门，70-80元，路程50分左右，人多建议打车，人少建议白云山集合路线； ## 上午十点前到达售票处 ### 轻松路线 #### 云台花园 10元门票 ![img](https://www.baiyunshan.com.cn/bys/jqdl/202003/97dd4f9a44464b479658d63917ef5935/images/ce5310273886404b9a197863916ff8b6.png) - [ ] 打卡赏析郁金香等，4月初可能还有一些花还在开； - [ ] 打卡中西合璧式园林，异国情调的喷泉，希望人少一些好拍照； - [ ] 打卡仙人掌 #### 白云索道 上山25元、下山20元 - [ ] 俯瞰景区内景点，远眺羊城 #### 山顶公园 - [ ] 打卡白云远望，羊城八景之一（可能傍晚看比较好吧，太早就变成了白云午望） - [ ] 山顶驿站据说有豆花和18蚊的文创雪糕进行商业打卡； #### 摩星岭 5元 ![img](https://www.baiyunshan.com.cn/bys/jqdl/202003/9ad634efea6d4c7391ad1a43dc784aac/images/df967b022b0d4048a91082c93a78754f.png) - [ ] 最高点，海拔382米（非常低的山岭），俯瞰广州 - [ ] 据说有个8蚊的豆花可以吃，休息一下 #### 白云晚望 ![img](https://www.baiyunshan.com.cn/attachment/userFile/20201218/20201218134917258.JPG) - [ ] 原路返回专门看白云远望，打卡广州塔、双塔，希望可以拍出爱乐之城的感觉 ### 值得绕路的景点打卡 由于景点内部错综复杂，所以可以前往不同地方，无法规划一条路线 #### 能仁寺 - [ ] 白云山中唯一保存完好的一座古寺庙，清咸丰同治年间修建，1993年广州市修复，不可燃烧香火； - [ ] 打卡“虎”字，台湾抗击倭寇，镇灾辟邪； - [ ] 打卡寺庙，慈云殿左侧文殊菩萨保佑论文录用； #### 毛利人运动文化村-蹦极、滑道等 - [ ] 值得挑战，但是**貌似最近倒闭**_(:з」∠)_； - [ ] 滑道貌似很长，可能还在营业 - [ ] 其他娱乐项目很奇怪，不介绍了 #### 天南第一峰 - [ ] 打卡“天南第一峰”牌坊，这是保存完好的古牌坊，宋代所建，后被重修； #### 广州碑林 5元 - [ ] 原为白云寺，抗日时被毁，92年重建； #### 白云飞索 - [ ] 号称全国最长，1500米，200元左右，各个平台价格很奇怪； #### 桃花涧 5元 - [ ] 仿照《桃花源记》设计的中式园林，据说有桃花，好像3月份中下旬开； - [ ] 感觉这个地方，适合中午吃饭休息，但是不建议景区吃饭，自带最好； #### 鸣春谷 10元 ![img](https://www.baiyunshan.com.cn/bys/jqdl/202005/1f17cca437dc4959b4c48d8d60663e35/images/4855f5d5d3444cfb8bd31f85d0baee93.png) - [ ] 我国目前最大鸟笼，天然式鸟笼； - [ ] 打卡大型鸟笼景区，孔雀、丹顶鹤、白天鹅和黑天鹅（没错，这些鹤和鹅是鸟类_(:з」∠)_）； - [ ] 打卡珍稀鸟展区，省鸟白鹇（很像一个大大的鸡），丹顶鹤； #### 九龙泉 - [ ] 听说很多广州人采集这里的水回家泡茶_(:з」∠)_； ## 下午两点左右下山 * 下山后吃饭，选择多，人流少，下山吃饭建议在山右侧或下侧吃（地铁附近的商业中心，吃完大概也是四点左右），距离学校近； ## 特殊调整 - [ ] **可以考虑先不去云台花园，直接上山，这样可以早点下来，比如一点左右，然后去南门山下吃饭（沙河粉村等等，附近有一些店，丰俭由人），吃完饭再去云台花园、麓湖公园等散步**。 ## 需要携带物品 - [ ] 身份证、本科生学生证（本科学历和60岁以上人群可以打折） - [ ] 背包； - [ ] 登山鞋； - [ ] 饮用水（每人至少两瓶）； - [ ] **驱蚊液**； - [ ] 干粮（水果、糕点、寿司等）； -
